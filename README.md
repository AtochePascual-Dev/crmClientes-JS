# CRM-Clientes

## Descripción

Este proyecto utiliza IndexedDB para administrar clientes, permitiendo agregar, actualizar y listar sus datos en una interfaz de usuario. Proporciona funciones para manejar la base de datos y mostrar alertas en caso de errores o acciones exitosas.
